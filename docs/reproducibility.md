# Reproducibility Guide

## Dataset: Comprehensive longitudinal dataset (1995-2023) on poverty rates across 30 countries in Sub-Saharan Africa, South Asia, East Asia, Latin America, MENA, and Europe/Central Asia. Variables: poverty rates at $1.90, $3.20, and $5.50/day thresholds, extrem

---

## 1. Requirements

- Web browser or data analysis software (R, Python, Excel)
- No special software required for basic data access

## 2. Data Access

```
# Clone repository
git clone https://github.com/juanmoisesd/poverty-rates-by-country-1995-2023-global-longitudinal-dataset.git
cd poverty-rates-by-country-1995-2023-global-longitudinal-dataset
```

## 3. Data Format

- Data is provided in standard CSV/HTML format
- UTF-8 encoding
- See `data_dictionary.csv` or README for variable descriptions
